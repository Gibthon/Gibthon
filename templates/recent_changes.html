Recent Changes

Welcome to the latest gibthon release, codenamed ''.
This page for letting you know what's been changed recently as well as what's in the works.

Changes and Improvements

  • Construct Designer

    Many improvements have been made to the designer. Fragments are now added and ordered using drag-and-drop, making designing constucts a breeze.

    Current Limitations

    • The designer currently produces information about the primers as well as warnings for things like misprimings. However, it is not currently possible to manually edit primer lengths, although this is something which is being worked on, see this issue.

    • Currently the designer only works with whole fragments. It is not possible to specify which sub-sequence you want to use in your fragments. A workaround is to use the fragment view to crop the fragment to the desired region. See this issue.

  • Fragment View

    A few mostly cosmetic improvements were made to the fragment view, particularly with the placement of features.

    One more notable change is that fragments can now be cropped to the current selection.

Known Bugs

Please visit our issue tracker on github to see the list of currently open bugs.
